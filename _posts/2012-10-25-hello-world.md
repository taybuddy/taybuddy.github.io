မင်္ဂလာပါ! ဒါကတော့ ကျွန်တော့်ရဲ့ Portfolio website ပေါ်မှာ ပထမဆုံး တင်လိုက်တဲ့ Blog post ဖြစ်ပါတယ်။

ဒီနေ့မှာတော့ ကျွန်တော့်ရဲ့ Website ကို **Jekyll** နဲ့ **GitHub Pages** သုံးပြီး ဘယ်လို တည်ဆောက်ခဲ့သလဲဆိုတာကို ပြန်လည်မျှဝေချင်ပါတယ်။ 

### ကျွန်တော် သင်ယူခဲ့ရတဲ့ အချက်များ:
* Git နဲ့ GitHub ကို ကျွမ်းကျင်စွာ သုံးတတ်လာခြင်း။
* Markdown (.md) နဲ့ စာရေးနည်းကို သိရှိလာခြင်း။
* SSH key သုံးပြီး ပိုမိုလုံခြုံအောင် ချိတ်ဆက်တတ်လာခြင်း။

နောက်ပိုင်းမှာလည်း ကျွန်တော့်ရဲ့ **Data Analytics** နဲ့ **Engineering** ဗဟုသုတတွေကို ဆက်လက်မျှဝေသွားပါဦးမယ်။
